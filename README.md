# Asistente de Predicación Bilingüe

Esta es una aplicación educativa diseñada para ayudar a predicadores a organizar temas bíblicos, registrar contactos y acceder a debates doctrinales. Está desarrollada en Python usando Kivy y ofrece soporte en **Español** e **Inglés**.

## Características principales

- Visualización de temas sugeridos aleatoriamente
- Artículos profundos por categoría
- Búsqueda por palabras clave
- Creación de usuario y login
- Selección de idioma
- Verificación de actualizaciones desde GitHub

## Requisitos

- Python 3.8 o superior
- Kivy 2.1.0 o superior
- Compatible con Ubuntu y Android (via Buildozer)
